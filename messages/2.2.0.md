# Sublime Rainmeter Release 2.2.0

This release adds smart auto completion. It offers context related auto completion.

* Smart Auto-Completion for:
    - `Meter`
    - `Measure`
    - `Plugin`
* Improved formula operations (e.g. `sin`, `cos`, etc.)
